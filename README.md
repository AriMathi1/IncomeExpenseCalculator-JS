Income & Expense Calculator

This is a web-based Income & Expense Calculator built using HTML, CSS, and JavaScript. The application allows users to manage their finances by adding, editing, deleting, and filtering income and expense entries. 
It provides a real-time overview of total income, total expenses, and net balance.


Core Functionality
- Add Transactions:
  - Input fields for adding income and expense entries with a description and amount.
- CRUD Operations:
  - Create: Add new entries.
  - Read: Dynamically display all transactions in a list.
  - Update: Edit existing entries directly in the list.
  - Delete: Remove entries from the list.
- Filters:
  - Radio button filters for viewing "All", "Income", or "Expense" entries.
- Reset Button:
  - Clears input fields for a seamless user experience.
- Financial Overview:
  - Displays total income, total expenses, and net balance at the top.
- Local Storage:
  - Ensures data persistence across sessions.
